Todo List App with Core Data - README

Introduction

This Todo List App helps you organize tasks easily. It stores data efficiently using Core Data.

Features

Add, Edit, Delete Tasks: Manage tasks effortlessly.
Persistent Storage: Your data is safe and accessible.

Requirements

Xcode
iOS device or simulator

Setup Instructions

Clone Repository: Get the code.
Open in Xcode: Launch the project.
Build and Run: Start the app.
Usage

Add Task: Tap "+" and enter task details.
Edit Task: Tap to modify.
Delete Task: Swipe left and tap delete.
